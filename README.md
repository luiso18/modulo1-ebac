Para que possa utilizar os arquivos devem seguir estes passos

1) deve se ter tanto o arquivo "calculadora1.py" quanto o "calculadora.sh"
   
2)é necessario que abra o terminal ubunto 22.04 para abrir o arquivo

3)os arquivos devem ser colocados no mesmo folder, por exemplo "/home/usuario/modulo1/atividade"

4)antes de se abrir o arquivo é preciso coloca-lo como executavel utilzando o seguinte metodo " chmod +x calculadora_sh"

5)apos faze-lo executavel use o comando "./calculadora_sh" para inicia-lo
